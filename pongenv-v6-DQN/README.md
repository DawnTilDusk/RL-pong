# PongEnv v6 (DQN)

本目录包含基于 Deep Q-Network (DQN) 的 Pong 游戏强化学习实现。此版本引入了神经网络来处理连续状态空间，并使用了经验回放等技术。

## 核心特点

1.  **连续状态空间**: 直接使用归一化后的球位置、速度和挡板位置作为输入，无需离散化。
2.  **深度神经网络**: 使用 PyTorch 构建全连接网络 (`DQNNetwork`) 拟合 Q 值函数。
3.  **经验回放 (Replay Buffer)**: 存储历史转移样本 `(state, action, reward, next_state, done)`，随机采样训练以打破相关性。
4.  **Target Network**: 使用目标网络计算 TD 目标值，提高训练稳定性。
5.  **实时统计 HUD**: 游戏界面左上角实时显示训练步数、探索率 (epsilon)、胜率、平均奖励和 Q 值均值。
6.  **可视化报表**: 训练结束或手动备份时，会自动生成训练曲线图表（需 `matplotlib`）。

## 实现原理

*   **状态定义 (`get_state`)**: 8 维向量
    *   球的 X/Y 速度（归一化）。
    *   左/右挡板 Y 速度（归一化）。
    *   球的 X/Y 坐标（归一化到 [-1, 1]）。
    *   左/右挡板 Y 坐标（归一化到 [-1, 1]）。
*   **网络结构**: 4 层全连接网络 (Input -> 64 -> 128 -> 64 -> Output)。
*   **训练流程**: 
    *   Epsilon-greedy 策略采样动作。
    *   将转移存入 Replay Buffer。
    *   从 Buffer 采样 Batch 进行梯度下降更新。
    *   定期同步 Target Network。

## 代码架构

*   `PongEnv.py`: 独立的环境类，处理物理模拟、碰撞检测和渲染。
*   `pong_bot_DQN.py`: 包含 `DQNNetwork` 定义、`Bot_DQN` 智能体逻辑以及主循环。
*   `dqn.pth`: 默认的模型保存文件。(删掉则从头开始训练，否则会从上次保存的模型继续训练)
    *   注意可以将history中的模型文件更名为`dqn.pth`，放置在当前目录下则会从该模型继续训练。
*   `history/`: 存放训练过程中自动备份的模型和图表。

## 快速开始

1.  确保已安装 `pygame`, `numpy`, `torch`, `matplotlib`。
2.  在当前目录下运行：
    ```bash
    python pong_bot_DQN.py
    ```
3.  **交互说明**:
    *   `F2`: 切换左侧挡板控制模式（玩家手动/简单的规则 Bot）。
    *   `F4`: 手动创建快照备份（保存模型 + 绘制曲线图）。
    *   退出程序时会自动保存当前模型。
