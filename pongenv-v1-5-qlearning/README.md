# PongEnv v1-5 (Q-learning)

本目录包含基于 Q-learning 的 Pong 游戏强化学习实现。这是最基础的版本，适合初学者理解强化学习的基本概念。

## 核心特点

1.  **离散状态空间**: 将连续的游戏画面（球的位置、速度，挡板位置）离散化为有限的状态网格。
2.  **Q-Table**: 使用多维数组（Q表）直接存储每个状态-动作对的价值（Q值）。
3.  **无神经网络**: 纯查表法，不依赖深度学习框架（PyTorch/TensorFlow）。
4.  **可视化训练**: 训练过程中实时渲染游戏画面，并直接在画面上绘制胜率和奖励趋势图。

## 实现原理

*   **状态定义 (`get_state`)**: 
    *   球的 X 方向速度（正/负）。
    *   球的 Y 方向速度（正/负）。
    *   左挡板与球的相对 Y 轴距离（离散化为 21 个区间）。
    *   右挡板与球的相对 Y 轴距离（离散化为 21 个区间）。
*   **动作空间**: 3个动作（上移、下移、不动）。
*   **奖励函数**: 
    *   击球奖励 (+200)。
    *   得分奖励 (+50)。
    *   失分惩罚 (-200)。
    *   辅助引导奖励（如靠近球、减少不必要的移动）。
*   **更新公式**: 标准 Q-learning 更新公式。
    $$ Q(s, a) \leftarrow Q(s, a) + \alpha [r + \gamma \max_{a'} Q(s', a') - Q(s, a)] $$

## 代码架构

*   `pongenv-v3.py`: 核心脚本，包含 `PongEnv` 环境类、`Bot_Q` 智能体类以及主训练循环。
*   `q_table-*.pkl`: 存储训练好的 Q 表文件。
*   `bg.png`: 游戏背景图。

## 快速开始

1.  确保已安装 `pygame` 和 `numpy`。
2.  在当前目录下运行：
    ```bash
    python pongenv-v3.py
    ```
3.  **交互说明**:
    *   程序默认运行 `bot_Q`（右侧智能体）进行训练。
    *   按 `F2` 可以保存当前的 Q 表（按时间戳保存，避免覆盖）。
    *   退出程序时会自动保存 Q 表到 `q_table-v3.pkl`。
